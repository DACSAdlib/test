Adlib - Notes and description control - Mathematical data

Requirement: As appropriate

Refer to:

• DACS 7.1 - Notes

With reference to format specific standards

Graphic materials - (Technical drawings and architectural plans)

• Refer to DCRM (G) 5B7 Technical drawings (Treat as drawings)

AND

• Refer to sections within BSR RDA MAP concerning scale and projection

• Where BSR RDA MAP instructions refer to cartographic materials, apply also to architectural plans

• Scale is required only when a scale statement is present on the unit of description

• Do not abbreviate when a projection statement is present on the unit of description
