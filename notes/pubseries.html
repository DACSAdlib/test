Adlib - Notes and description control - Publisher Series

The purpose of this element is to provide the title and statement of responsibility of a publisher's series, where applicable.

FIELD IS PROBLEMATIC

Requirement: See DQSS prior to use

This field is not for entries concerning publications about the unit of description

In RAD2 there is an entire section: 1.6 Publisher's series area

In the ACMS draft manual Publisher's Series is treated as a 'Specialized Element'

There is no equivalent instruction regarding this area in DACS, aside from a single reference in the Notes area introduction.
